Flight Ticketing System

It is a simple Flight Ticketing System where the user can reserve and buy tickets. The admins can also manage the destinations by editing the flights. Sales report is also issued since it serves as the receipt of the flight they booked.

Movie Ticketing System

It is a simple Movie Ticketing System where the user can check the list of showing movies and buy tickets. It also issues a receipt as a proof that you already bought a ticket for a movie.

Online Bakery Shop System

It is an Online Bakery Shop System wherein the users are able to register, log in, view all the the products, add to cart, and also purchase.

Coffee Flyer

It is a flyer for a coffee shop named Hilltop Cafe. This flyer is printed in 1 page only with a back to back design which includes the menu of the cafe.
